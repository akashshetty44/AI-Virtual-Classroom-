Project Overview

AI Virtual Classroom is an Artificial Intelligence based smart learning platform designed to automate classroom activities and improve student engagement.

👁 Computer Vision

Face recognition attendance and emotion detection using OpenCV and DeepFace.

🧠 Natural Language Processing

Automatic lecture transcription, summarization, and quiz generation.

📊 Machine Learning

Predicts student performance using academic data.


Technologies Used

  • Python
  • Flask
  • OpenCV
  • DeepFace
  • Whisper AI
  • Hugging Face Transformers
  • Scikit-learn
  • SQLite
